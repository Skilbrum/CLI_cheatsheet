# Git

### Update modified files to local index
```
git add -u
```
### Add all files to local index
```
git add --all
```

### Local ommit with message
```
git commit -m "message"
```
### Push to remote
```
git push origin branchname
```

### Discard all changes
```
git checkout -- '*'
```

### Stash changes
```
git stash
```

### Stash list
```
git stash list
```
